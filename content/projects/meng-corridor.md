## 关于这个项目

梦回廊是我个人使用 Next.js 16 + Tailwind CSS 搭建的个人网站。

### 主要功能

- 樱花飘落动画
- MDX/Markdown 博客系统
- 项目展示
- 一言 API 集成
- 二次元主题背景

### 技术栈

- **框架**: Next.js 16 (Turbopack)
- **样式**: Tailwind CSS v4
- **内容**: Markdown + frontmatter
- **部署**: Cloudflare Pages
