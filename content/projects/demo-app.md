## 项目简介

Demo App 是一个学习 SpringBoot 时搭建的后端项目，涵盖常见的 CRUD 操作和 RESTful API 设计。

### 实现的功能

- 用户注册与登录
- JWT 认证
- 文章增删改查
- 分页查询

### 学到了什么

- SpringBoot 自动配置原理
- MyBatis 的 XML 映射
- RESTful 最佳实践
